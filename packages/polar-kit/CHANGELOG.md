## [0.1.0] - 2025-12-11

### Added
- Initial release of @makeco/polar-kit CLI for Polar
- Product and price management (create, archive, update)
- Database synchronization with adapter pattern
- Multi-environment support (test, dev, staging, prod)
- User preferences storage and persistence

### Changed
- Simplified package structure with flattened file organization
- Streamlined codebase for Polar-specific functionality
- Consolidated command files into singular modules
